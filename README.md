# git-study

# 基础操作
```bash git init:初始化本地库

git add 文件名|通配符:添加至暂存区

git commit -m "提交版本信息":提交暂存区至本地库

git rm --cached 文件名|通配符:删除暂存区文件

git status:查看本地库状态

git reflog:查看精简日志信息

git log:查看详细日志信息

git reset --hard 版本号:版本切换
```
# 分支操作
```bash git branch 分支名:创建分支

git branch -v:查看分支

git checkout 分支名:切换分支

git merge 分支名:将指定的分支合并到当前分枝上
```
